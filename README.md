# 🚆 Online Reservation System (Java)

## 📌 Project Overview
The Online Reservation System is a desktop application developed using Java Swing and MySQL. It allows users to reserve train tickets, search reservations, cancel tickets, and view all reservations through a user-friendly interface.

## 🚀 Features
- User Login
- Reserve Train Ticket
- Fetch Train Details
- Search Reservation by PNR
- View All Reservations
- Cancel Reservation
- Input Validation
- MySQL Database Connectivity

## 🛠️ Technologies Used
- Java
- Java Swing
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub

## 📂 Project Structure
```
src/
 ├── database/
 ├── model/
 ├── service/
 ├── ui/
 └── Main.java
```

## ▶️ How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Configure the MySQL database.
4. Run `Main.java`.
